🧠 AI Developer Assistant (Code Debugger + Reviewer)

An AI-powered web application built with Python and Streamlit that helps developers debug, improve, and learn from their code. Powered by Google's Gemini API, this tool acts as both a sharp-eyed bug hunter and a friendly programming teacher.
🚀 Features
💻 Multi-Language Support: Choose from Python, JavaScript, Java, C++, SQL, HTML/CSS, and more.
🔧 Debug Mode: Automatically detects errors in your code, explains the issue in simple terms, and provides the corrected code.
🌟 Review Mode: Acts as a "Friendly Programming Teacher" to review your code quality, give you a score, and offer beginner-friendly tips on how to improve readability and structure.
⚡ Both Mode: Get a full diagnostic—debugging and a code review at the same time.
🎨 Clean UI: Built with Streamlit for a smooth, responsive, and easy-to-use interface.
🛠️ Tech Stack

Frontend: Streamlit
AI Model: Google Gemini API (gemini-3-flash-preview)
Language: Python
Environment: python-dotenv for secure API key management
